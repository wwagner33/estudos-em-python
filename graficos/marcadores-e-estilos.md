## Formatos de marcadores disponíveis para plot():
'o' : círculo

's' : quadrado

'^' : triângulo para cima

'v' : triângulo para baixo

'<' : triângulo para esquerda

'>' : triângulo para direita

'D' : diamante

'p' : pentágono

'*' : estrela

'+' : cruz

'x' : x

## Estilos de linha:
'-' : sólida

'--' : tracejada

':' : pontilhada

'-.' : traço-ponto
